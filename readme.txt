WhatsTrack is an abbreviation consisting of Whats (from Whatsapp) and Track (from Google Event Tracking). Integrate Whatsapp with your website and track visitor clicks using Google event tracking.

== Description ==

You will be able to add a vertically scrolling Whatsapp button on the bottom right side of your web page. Add your Whatsapp number as well as any intro text on the settings page so that the visitor can contact you regarding their query.
The image provided with this plugin is FREE and you can use it freely on your website.
Please go through the https://www.adeelsarfaraz.com/terms-of-service/ and https://www.adeelsarfaraz.com/privacy-policy-website/ for more details regarding usage of this plugin on your website.

== Installation ==

Upload the WhatsTrack plugin to your blog, activate it, enter your Whatsapp ID (number registered on Whatsapp) as well as any intro text and the Whatsapp image will start displaying on the bottom right side of your website. That's all there is to it. 

== Frequently Asked Questions ==

1. Which version of Google Analytics should I use?
You should use Google Tag Manager. Its the latest tool provided by Google. You can read more about it here https://stackoverflow.com/questions/23640645/what-is-the-difference-between-google-tag-manager-and-google-analytics

2. Do I need to use the image you have provided in the plugin or can I replace it with mine?
The plugin provides you the image by default and its FREE. You can use it freely on your website however if you want to replace it with another image you can do that as well. Just keep the name of the image same so that the plugin picks up that image on the next page load.
